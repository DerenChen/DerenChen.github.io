# Core Bluetooth
与低功耗蓝牙 4.0设备通讯
-----------------


## 概述
`Core Bluetooth`框架提供了`app`与设备了低功耗蓝牙设备通讯的功能丰富的类。
不要继承任何`Core Bluetooth`框架提供的类。重载这些类是不推荐的，而且后果是不堪设想。

> 重点
> iOS app在`iOS 10.0`后，需要在 `Info.plist`中配置权限：`NSBluetoothPeripheralUsageDescription`。
